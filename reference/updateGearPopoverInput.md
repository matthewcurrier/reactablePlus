# Update a gear popover from the server

Update a gear popover from the server

## Usage

``` r
updateGearPopoverInput(
  session = getDefaultReactiveDomain(),
  inputId,
  value = NULL
)
```

## Arguments

- session:

  The Shiny session object.

- inputId:

  Character. The input ID to update.

- value:

  Named list of toggle values to update.

## Value

Called for its side effect; returns invisibly.
