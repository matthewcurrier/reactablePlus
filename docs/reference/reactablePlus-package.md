# reactablePlus: Enhanced Interactive Tables with Inline Editors for Shiny

Extends 'reactable' with inline editing widgets (pickers, popovers, text
inputs) and a declarative configuration layer for building editable
tables in Shiny applications. Provides a generic editable table module
driven by row and column specifications, plus a library of cell-level
input widgets with custom Shiny bindings.
